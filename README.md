## Power GPT

OpenAI in your command prompt



### Getting Started 

Requires nodejs 16 (can install with nvm) 

1. Create a file named .env and add the token OPENAI_API_KEY= followed by your api key  (https://platform.openai.com/account/api-keys)

2. yarn && yarn start
